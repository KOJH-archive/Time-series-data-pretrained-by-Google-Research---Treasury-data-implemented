# Google TimesFM 기반 미국채 금리차(Spread) 예측 파이프라인

## 1. 프로젝트 개요
이 프로젝트는 Google Research의 **TimesFM 2.5** 파운데이션 모델을 활용하여 미국 국채 금리차(10Y-2Y, 30Y-10Y)를 분석하고 향후 추이를 예측하는 자동화 파이프라인입니다.

## 2. 주요 기능
- **데이터 수집 (`fetch_treasury_v2.py`)**: FRED(Federal Reserve Economic Data) API를 통해 최신 미국채 금리 데이터를 수집하고 금리차(Spread)를 계산합니다.
- **시계열 예측 (`predict_input.py`)**: `google/timesfm-2.5-200m-pytorch` 모델을 사용하여 수집된 금리차 데이터의 향후 추이를 예측합니다.
- **리포트 생성 (`generate_report.py`)**: 예측 결과를 분석하여 현재 상태 및 향후 전망을 담은 Markdown 리포트를 자동으로 생성합니다.
- **원클릭 실행 (`TimesFM_실행하기.bat`)**: 데이터 수집부터 예측, 리포트 생성까지의 전 과정을 한 번에 수행할 수 있는 배치 파일을 제공합니다.

## 3. 기술 스택
- **Model**: TimesFM 2.5 (200M parameters, PyTorch)
- **Language**: Python 3.10+
- **Libraries**: `torch`, `timesfm`, `pandas`, `matplotlib`, `koreanize-matplotlib`

## 4. 디렉토리 구조
- `input_data/`: 수집된 금리차 CSV 데이터 저장
- `output/`: 예측 결과 그래프(PNG), 예측 데이터(CSV) 및 최종 분석 리포트(MD) 저장
